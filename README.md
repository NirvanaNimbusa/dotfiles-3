dotfiles by [@kevinji](https://github.com/kevinji).

## Installation
Clone this repo anywhere:
```bash
git clone --recursive https://github.com/kevinji/dotfiles.git
```

Then, run `./install`, followed by `./scripts/setup-mac.sh` if relevant.
