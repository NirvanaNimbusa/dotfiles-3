dotfiles by [@mc10](https://github.com/mc10).

## Installation
Clone this repo anywhere:
```bash
git clone --recursive https://github.com/mc10/dotfiles.git
```

Then, run `./install`, followed by `./scripts/setup-mac.sh` if relevant.
